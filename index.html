     Circle of Life

    A single-player game created with G develop and deployed on Netlify, in which you are pursued by multiple enemies and must avoid colliding with them. The difficulty level of the game increases as the player keep scoring. The score depends on the time length you survive and enemies you kill before dying. The white light is the proximity area using which the enemies can sense and start pursuing you. The player has to kill the enemies by dodging and drifting them into themselves. The controls are simple four arrow keys on the keyboard.

    • Link: https://circlelife.netlify.app/

     Fake news Detection

    A BERT based tool to classify a piece of text into fake or real, implemented using Simpletransformers and Huggingface. Our approach takes the stance detection, text and emotional aspect of given text into consideration. It takes the text as input and comparing its stance with various trustable resources available online. 

       AI for an eye

      An easy to use AI enabled tool for detection of ocular disease. After giving a fundus image as input the classifier can predict whether the image is of left or right eye. It further tells whether it is a normal eye or it is suffering from any eye disease like Cataract, Glaucoma, hypertension, Myopia and AMD. 

      • Link: https://fundus.anvil.app/
      • Github: https://github.com/nimbolkarshravani/AI-for-an-eye

       SensolAI

      SensorAI is a tool to detect the activity of user using the timeseries data coming from smart phone sensors including accelerometer and gyroscope sensors (attitude, gravity, user Acceleration, and rotation rate). The deep neural network can detect activities like walking, standing, sitting, upstairs, downstairs etc. The model is trained on sensor data of IPhone6s. 

      • Github: 
        https://github.com/nimbolkarshravani/SensorAI/tree/main

       AgroML

      A Crop weed detection system for identifying the unwanted plants called weeds from the soybean crop. The classifier neural network can identify the weed, crop and soil from a given image. This model can be used in the system where we want to automate the task of spraying weedicides and also save the crops from unwanted spraying of harmful chemicals.

      • Link: https://agroml.netlify.app/
      • Github: https://github.com/nimbolkarshravani/AgroML
